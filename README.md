# 🚀 Droid Translator

Droid Translator — это лёгкий и быстрый веб-переводчик с поддержкой более 100 языков.
Приложение работает прямо в браузере и ориентировано на простоту, скорость и удобство использования.

---

# 🌟 Возможности

- 🌍 Перевод текста на 100+ языков
- 🔄 Автоопределение языка
- 🔊 Озвучка текста (Text-to-Speech)
- 📋 Быстрое копирование перевода
- ⚡ Минималистичный и быстрый интерфейс
- 📱 Удобство использования на мобильных устройствах

---

# 🛠️ Технологии

- HTML5
- CSS3
- JavaScript (Vanilla)
- Tailwind CSS
- Google Translate API (неофициальный)
